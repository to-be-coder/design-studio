# Studio Wiki — index

One line per page, grouped by entity. Maintained by `design-studio-harvest` on every crossing;
humans may edit freely — agents must keep it current.

## Patterns
- [[Specimen boards]] — judge candidate visual languages from rendered boards generated from the
  tokens themselves; never ask a human to judge YAML.

## Plays
- [[Single copy moving home]] — move an artifact to where it's enforced and leave a link behind;
  never mirror-and-sync.
- [[Derive before invent]] — codify an existing identity faithfully before proposing anything new.
- [[One contract for parallel agents]] — fan-out stays consistent only when every agent reads the
  same small law first.

## Traps
- [[Abstract formats die on contact]] — formats authored from memory or in the abstract fail
  against real material; pressure-test before automating.

## Standards
- [[design-md essentials]] — the design.md format: fixed sub-token vocabulary, fixed section
  order, lint/export/diff CLI, v0.3.0 gotchas.
